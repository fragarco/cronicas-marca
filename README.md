# MODULO DE CONTENIDO CRÓNICAS DE LA MARCA PARA FOUNDRY VTT

Contenido del juego de rol Crónicas de la Marca del Este para Foundry VTT. Este módulo require del sistema Castles and Crusades para poder funcionar adecuadamente. En concreto contiene:

- Habilidades
- Equipo
- Conjuros
- Monstruos
